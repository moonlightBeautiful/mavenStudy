#mavenStudy
一些资源文件(xml、properties等)，可能打包不进去。所以可以在build标签下添加：
    <resources>
        <resource>
            <directory>src/main/java</directory>
            <includes>
                <include>**/*.properties</include>
                <include>**/*.xml</include>
            </includes>
        </resource>
        <resource>
            <directory>src/main/resources</directory>
            <includes>
                 <include>**/*.properties</include>
                 <include>**/*.xml</include>
            </includes>
        </resource>
    </resources>
